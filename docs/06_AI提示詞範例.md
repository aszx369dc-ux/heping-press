# 06 AI 提示詞範例

下面每一段都可以直接複製，再依照實際情況修改書名、檔名或需求。

每一個 Prompt 都已經強調：

- 只修改指定檔案
- 不要重寫整支檔案
- 不要順便優化
- 不要 Commit
- 不要 Push
- 完成後提供 git diff

## 1. 請 ChatGPT / Gemini 整理目錄

```text
請幫我整理這本書的故事目錄。
我會貼上每篇故事的標題、作者與起始頁。

請只幫我整理成 data.js 可用的 stories 陣列格式。
不要修改任何專案檔案。
不要猜頁碼。
不要順便優化。
```

## 2. 請 Codex 新增一本書

```text
請只修改 data.js、index.html、script.js。
不要修改 style.css。
不要重寫整支檔案。
不要順便優化。
不要 Commit。
不要 Push。

目標：
新增一本新書，名稱是《家庭故事集》。
英文資料夾名稱是 family-stories。
JavaScript id 是 familyStories。

請完成後提供 git diff。
```

## 7. 請 AI 檢查 git diff

```text

```
